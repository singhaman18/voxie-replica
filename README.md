## ğŸš€ Quick Start

### **Prerequisites**
- **Node.js** v18.0.0 or higher
- **npm** v8.0.0 or higher (or **yarn**/**pnpm**)
- Modern browser with **WebGL support**

### **Installation**

```bash
# 1. Clone the repository
git clone https://github.com/devanshuK45/Cmdhub.git
cd Cmdhub

# 2. Install all dependencies (includes 3D graphics libraries)
npm install

# 3. Start development server with hot reload
npm run dev

# 4. Open browser at http://localhost:8081
```

### **Production Build**

```bash
# Build optimized production bundle
npm run build

# Preview production build locally
npm run preview

# Serve at http://localhost:4173
```
